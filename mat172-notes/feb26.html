Multiple Angle Equations & Chapter 5 Review

Exam Logistics: The Chapter 5 test is next Tuesday. This is our first paper-and-pencil test. There are 19 questions. It is generally shorter than the practice test because you aren't forced to do "Verify" questions exactly how the software wants. You will be provided with a formula sheet containing Chapter 4 and 5 material and a blank unit circle to doodle on.

BIG IDEA

Solving equations with multiple angles (e.g., $\tan(3x)$ or $\sin(\frac{2}{3}\theta)$) requires starting with the "all solutions" scenario. Because the period is compressed or stretched, you must find all general solutions first, then divide by the coefficient to find specific solutions within the $[0, 2\pi)$ interval. This often results in a significantly higher number of solutions than standard equations.

KEY TERMS

  • Multiple Angle: An equation where the variable inside the trig function is multiplied by a constant (e.g., $nx$).
  • All Solutions Notation: Adding $+ 2\pi n$ (for sine/cosine/secant/cosecant) or $+ \pi n$ (for tangent/cotangent) to account for all periodic rotations.
  • Redundancy: In tangent equations, because the period is $\pi$, solutions in Quadrant III are often covered by the $+ \pi n$ notation added to the Quadrant I solution.
  • LCD: Lowest Common Denominator, essential for adding fractional pi increments when solving for $x$.

STEPS/PROCESS

  1. Isolate the Trig Part: Use basic algebra to get the trig function by itself.
  2. Identify Unit Circle Values: Determine where the trig function equals the given value within the first revolution, ignoring the multiple angle coefficient for a moment.
  3. Write General Solutions: Tack on $+ 2\pi n$ or $+ \pi n$.
  4. Divide/Multiply: Isolate the variable by dividing everything (including the $n$ term) by the coefficient.
  5. Plug in Chunks (P&C): Substitute $n = 0, 1, 2...$ until your solutions exceed the $2\pi$ (or specified) limit.

EXAMPLE 1: Multiple Angle Tangent

Problem: Solve $\tan(3x) = \frac{\sqrt{3}}{3}$ for $[0, 2\pi)$.

1. Identify the base angle: $\tan$ is $\frac{\sqrt{3}}{3}$ at $\frac{\pi}{6}$ (Quadrant I) and $\frac{7\pi}{6}$ (Quadrant III).

2. Set up the equation for all solutions: $$3x = \frac{\pi}{6} + \pi n$$ (Note: $\frac{7\pi}{6}$ is redundant because $\frac{\pi}{6} + \pi = \frac{7\pi}{6}$).

3. Solve for $x$ by dividing by 3: $$x = \frac{\pi}{18} + \frac{\pi}{3}n$$

4. Plug in $n$ values (LCD is 18, so $\frac{\pi}{3}n = \frac{6\pi}{18}n$):

  • $n=0: \frac{\pi}{18}$
  • $n=1: \frac{7\pi}{18}$ (pattern: add $6\pi$ to the numerator)
  • $n=2: \frac{13\pi}{18}$
  • $n=3: \frac{19\pi}{18}$
  • $n=4: \frac{25\pi}{18}$
  • $n=5: \frac{31\pi}{18}$
  • $n=6: \frac{37\pi}{18}$ (STOP: This is $> 2\pi$, as $2\pi = \frac{36\pi}{18}$)
Final Solutions: $\frac{\pi}{18}, \frac{7\pi}{18}, \frac{13\pi}{18}, \frac{19\pi}{18}, \frac{25\pi}{18}, \frac{31\pi}{18}$.

EXAMPLE 2: Fractional Multiple Angle (No Solution)

Problem: Solve $\sin(\frac{2}{3}\theta) = -1$ for $[0, 2\pi)$.

1. Base angle: $\sin$ is $-1$ at $\frac{3\pi}{2}$.

2. Set up equation: $\frac{2}{3}\theta = \frac{3\pi}{2} + 2\pi n$.

3. Isolate $\theta$ by multiplying by the reciprocal ($\frac{3}{2}$): $$\theta = \frac{9\pi}{4} + 3\pi n$$

4. Check interval: $\frac{9\pi}{4}$ is $2.25\pi$, which is already greater than the $2\pi$ limit. Adding or subtracting $3\pi n$ will not bring the value into the $[0, 2\pi)$ range.

Result: No Solution / Empty Set ($\emptyset$).
Calculator Check Method: Move everything to one side so the equation equals zero. Graph the function in your calculator. Set your window $Xmin = 0$ and $Xmax = 2\pi$. Look for X-intercepts. To validate exact values, set your $Xscale$ to the common denominator (e.g., $\pi/18$) and count the tick marks, or use the Table set to $\Delta Tbl = \pi/18$.

RULES/IDENTITIES FOR REVIEW

  • Quadratic Trig Equations: If an equation looks like $a(\tan^2 x) + b(\tan x) + c = 0$, you must factor it or use the Quadratic Formula: $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$.
  • Radical Restrictions: Most textbooks/software allow radicals inside radicals but do not allow fractions under radicals or radicals in the denominator.
  • Verifying Identities: Trust your instincts. If a software says you are wrong but your math is sound, check the decimal approximation in a calculator. If they match, your version is likely just a different algebraic form.

PRACTICE TIPS

1. Study Plan: Use the problem codes provided in the Canvas outline (e.g., 5.3.51) to find similar problems in the MyMathLab study plan.

2. Formula Usage: If a problem specifies "Use the Half-Angle Formula," you must use it. Otherwise, there are often multiple valid paths to the same answer.

3. Trace Tool: When checking on a calculator, the trace tool helps you see if the graph actually touches the X-axis or just gets close (important for "No Solution" cases).

SUMMARY

Success in Chapter 5 involves recognizing unit circle values, mastering the algebraic rearrangement of trig functions, and being meticulous with periodic additions ($2\pi n$ vs $\pi n$) in multiple-angle problems. For the upcoming test, focus on clear scratch work and validating your results against the specified interval $[0, 2\pi)$.
