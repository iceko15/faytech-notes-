MAT-172: Solving Trig Equations & Inverse Functions

BIG IDEA

When solving trigonometric equations, using an inverse function on a calculator will only provide a single output (the principal value). Because trigonometric functions are periodic, they often have an infinite number of solutions. We must interpret the calculator's output as a reference angle or a specific quadrant value and then apply our knowledge of the unit circle and periodic properties to identify all valid solutions within a given interval or for all real numbers.

KEY TERMS

  • Inverse Trig Function: A function used to find an angle when the trig value is known; however, it only returns one value within a restricted range.
  • Infinite Solutions: Most trig functions have solutions that repeat every revolution ($2\pi$ or $360^\circ$).
  • Reference Angle: The acute angle formed with the x-axis, used to find solutions in different quadrants.
  • Radians vs. Degrees: 1 radian is approximately $57^\circ$. If you are stuck in radians, multiply by $\frac{180}{\pi}$ to check your quadrant.

STEPS/PROCESS

  1. Isolate the Trig Function: Get the $\sin$, $\cos$, or $\tan$ term by itself.
  2. Apply Inverse Function: Use the inverse (e.g., $\sin^{-1}$) to get the initial angle.
  3. Identify the Quadrant: Look at the sign of the trig value to determine which quadrants the solutions live in.
  4. Determine All Solutions:
    • For specific intervals (e.g., $[0, 2\pi]$), find all angles in that range.
    • For "all solutions," add the period multiple: $+\, 2n\pi$ (for sine/cosine) or $+\, n\pi$ (for tangent).
  5. Adjust for Multiple Revolutions: If the inverse function gives a value like 1.24 radians, treat it as a reference and tweak based on the desired quadrant.

RULES/IDENTITIES

To represent all possible solutions across infinite revolutions:

$$\theta = \text{Initial Angle} + 2k\pi$$

Where $k$ (or $n$) represents any integer, accounting for every rotation around the unit circle.

Calculator Strategy: When in doubt, punch the radian value into the calculator and convert to degrees to "eyeball" the location. If you have $1.24$ radians, it is a smidge over $57^\circ$, so it must be in Quadrant I.

EXAMPLE 1: Calculator Interpretation

Scenario: You use an inverse function and the calculator outputs a single number, but the problem asks for all solutions in $[0, 2\pi]$.

Process: If the calculator gives $1.24$, and you know the cosine is positive, you must also find the solution in Quadrant IV. You would calculate $2\pi - 1.24$.

SUMMARY

The inverse trig functions are limited by their range. To solve equations fully, you must: 1. Use the calculator value as a starting point. 2. Use the unit circle to find the second location where that trig value exists. 3. Add $2n\pi$ for general solutions. 4. Always double-check your quadrant by converting to degrees if radians feel "fuzzy."
