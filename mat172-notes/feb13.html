Feb 13: Trigonometric Graphs & Chapter 4 Test Review

BIG IDEA

Success in Chapter 4 requires a synthesis of right-triangle trigonometry (SOH-CAH-TOA), understanding trigonometric transformations (amplitude, period, and phase shifts), and the ability to verify manual sketches against calculator outputs. Key focus is placed on "letting the numbers prove the quadrant" rather than relying on potentially inaccurate hand-drawn diagrams.

Exam Warning: I tend to overwrite my answer keys so that you can see exactly where every number comes from. These are not magic numbers; they are derived from specific information we have covered. Pay attention to coterminal angles in both degrees and radians—sometimes one helps you spot-check the other.

KEY TERMS

  • SOH-CAH-TOA: The foundational tool for solving right triangles and practicing trigonometric ratios.
  • Quadrantal: When a calculation falls on the border between two quadrants; trust the numerical result over a visual sketch.
  • Reciprocal Functions: Cosecant ($\csc$), Secant ($\sec$), and Cotangent ($\cot$). These are graphed in calculators using the $1/\text{function}$ relationship.
  • Reference Angle: The acute angle formed with the x-axis, used to identify the values in Quadrant 1 and then adjusted for the terminal side's actual quadrant.
  • Phase Shift: The horizontal shift of a trigonometric graph.

STEPS/PROCESS

Graphing Cosecant ($\csc$) and Secant ($\sec$)

  1. Identify the Reciprocal: For $y = 2\csc(x + \pi/4)$, first identify the corresponding sine function: $y = 2\sin(x + \pi/4)$.
  2. Map Landmarks: Determine where the original (transformed) sine function equals zero. These locations become the vertical asymptotes for the cosecant graph.
  3. Identify Max/Min: The relative maximums of the sine curve become the relative minimums of the cosecant curves, and vice versa.
  4. Calculator Verification: Since most calculators lack a $\csc$ button, input the function as: $$y = \frac{2}{\sin(x + \pi/4)}$$ Note: The coefficient (2) stays in the numerator; only the trig portion is reciprocated.
  5. Window Settings: To check your work on a calculator, manually set the window to match your graph paper scale (e.g., $X\text{min} = -\pi$, $X\text{max} = 2\pi$, $X\text{scl} = \pi/4$).
Instructor Insight: Don't forcibly convert everything to degrees. You need a working knowledge of radians. If you are stuck, you can check your work in degrees, but you must be comfortable with values like $7\pi/6$.

RULES/IDENTITIES

  • Asymptote Notation: Use the "equals" sign to denote the location of an asymptote (e.g., $x = -\pi/4$), but understand it represents where $x$ cannot be.
  • Nth Version of Asymptotes: Asymptotes repeat. You can start with the first positive asymptote or a negative one; they represent the same set of values. Some textbooks prefer starting with the first positive one for "better grammar."
  • Reference Angle Rule: Determine the quadrant where the angle terminates to decide if the value is positive or negative. Use the Quadrant 1 reference angle to find the numerical value.

EXAMPLE 1: The Isosceles Shortcut

Problem: Solving for a missing side in a triangle (Number 47).

The "Long" Way: Using standard SOH-CAH-TOA procedures without noticing triangle properties.
The "Safe" Way: If a triangle is not isosceles, you must use the full trigonometric method to find the value.
The Efficiency Tip: As pointed out by Reese in class, if you notice the triangle is isosceles, the value might be determined in essentially one step. However, always be prepared to use the "fluid" trigonometric method if the triangle is scalene.

EXAMPLE 2: Navigational Bearings & Quadrant Proof

Scenario: A diagram suggests a bearing might be in Quadrant 4, but calculations result in an angle of $87^\circ$ from the North-South line.

Logic:
  • If the angle is $87^\circ$, it is still within the $90^\circ$ limit of Quadrant 1.
  • If the calculation had exceeded $90^\circ$, it would have "trickled down" into Quadrant 4, requiring a change in notation to "South and East."
  • Moral: Don't fixate on your picture if it isn't perfectly to scale. Let the numbers prove the quadrant.
