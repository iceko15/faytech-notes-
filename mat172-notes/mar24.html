Complex Roots & Vector Fundamentals

BIG IDEA

The transition from complex numbers to vectors involves using De Moivre's Theorem to find multiple roots of a complex number and understanding that vectors represent magnitude and direction, regardless of their starting position in space.

KEY TERMS

  • Principal Root: The primary real-valued solution (e.g., $x=2$ for $\sqrt[3]{8}$).
  • Magnitude: The size, length, or "strength" of a vector, denoted by double bars $||V||$.
  • Standard Position: A vector reoriented to begin at the origin $(0,0)$.
  • Resultant: The sum of two or more vectors.
  • Scalar: A plain number used to multiply a vector, affecting its size or direction.
  • Unit Vector: A vector with a magnitude of exactly 1, maintaining the same direction as the original.

STEPS/PROCESS

Finding Complex Roots

  1. Convert the number to polar form ($r$ and $\theta$).
  2. Identify $n$ (the root being sought) and the $k$ values ($k = 0, 1, 2, ... n-1$).
  3. Plug into the root formula: $z_k = \sqrt[n]{r} [\cos(\frac{\theta + 360k}{n}) + i\sin(\frac{\theta + 360k}{n})]$.
  4. Calculate each root by iterating $k$.
Professor's Shortcut: Use the "Second + Entry" (Last Entry) function on your calculator to quickly swap $k=0$ for $k=1, 2, 3$ without retyping the entire string.

Converting Vectors between Points

If given initial point $(x_1, y_1)$ and terminal point $(x_2, y_2)$:

  • $a = x_2 - x_1$
  • $b = y_2 - y_1$
  • Vector $v = ai + bj$

RULES/IDENTITIES

De Moivre's Theorem for Roots

$$W_k = \sqrt[n]{r} \left( \cos \frac{\theta + 2\pi k}{n} + i \sin \frac{\theta + 2\pi k}{n} \right)$$

Vector Magnitude

$$||v|| = \sqrt{a^2 + b^2}$$

Unit Vector Formula

$$u = \frac{v}{||v||}$$

EXAMPLE 1: Complex Cube Roots

Problem: Find the complex cube roots of $z = 8(\cos 210^\circ + i\sin 210^\circ)$.
Parameters: $n=3$, $r=8$, $\theta=210^\circ$. $k=0, 1, 2$.
  • For k=0: $\sqrt[3]{8} = 2$. Angle: $\frac{210 + 360(0)}{3} = 70^\circ$.
    $z_0 = 2(\cos 70^\circ + i\sin 70^\circ)$
  • For k=1: Angle: $\frac{210 + 360(1)}{3} = \frac{570}{3} = 190^\circ$.
    $z_1 = 2(\cos 190^\circ + i\sin 190^\circ)$
  • For k=2: Angle: $\frac{210 + 360(2)}{3} = \frac{930}{3} = 310^\circ$.
    $z_2 = 2(\cos 310^\circ + i\sin 310^\circ)$

EXAMPLE 2: Vector from Two Points

Problem: Initial point $P = (-4, -4)$, Terminal point $Q = (6, 2)$. Find vector $v$.
Calculation:

$a = 6 - (-4) = 10$

$b = 2 - (-4) = 6$

Result: $v = 10i + 6j$

Magnitude: $||v|| = \sqrt{10^2 + 6^2} = \sqrt{136} \approx 11.66$

EXAMPLE 3: Resultant Force

Problem: Force 1: 30 lbs at $N 10^\circ E$. Force 2: 60 lbs at $N 60^\circ E$.
Step 1: Convert to Standard Angles
  • $\theta_1 = 90 - 10 = 80^\circ$
  • $\theta_2 = 90 - 60 = 30^\circ$
Step 2: Component Addition

$A_{total} = 30\cos(80^\circ) + 60\cos(30^\circ) \approx 5.21 + 51.96 = 57.17$

$B_{total} = 30\sin(80^\circ) + 60\sin(30^\circ) \approx 29.54 + 30.00 = 59.54$

Step 3: Magnitude

$||R|| = \sqrt{57.17^2 + 59.54^2} \approx 82.54 \text{ lbs}$

Exam Hint: On the test, you must present the number of answers corresponding to the root (e.g., 4th root requires 4 answers). MyMathLab prompts you for the number of slots, but the paper test will not.

PRACTICE PROBLEMS

  1. Find the square roots of $16i$. (Hint: $r=16, \theta=90^\circ$).
  2. Calculate the unit vector for $v = 4i - 3j$.
  3. A plane flies at 200 mph at $S 30^\circ W$. Write its velocity vector in $i, j$ form.

SUMMARY

Complex roots are found by distributing a full circle ($360^\circ$) across $n$ solutions. Vectors represent the "push" or "pull" of a force; adding them (resultant) requires breaking them into horizontal ($i$) and vertical ($j$) components, summing those components, and then using the Pythagorean theorem to find the final magnitude.
