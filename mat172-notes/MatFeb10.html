BIG IDEA

Inverse trigonometric functions are used to find angles. They take a ratio as input and return an angle as output. The crucial concept is that their ranges (outputs) are restricted to specific quadrants to ensure there is only one unique answer. This is different from solving a standard trig equation like $\sin(x) = 1/2$, which has infinite solutions. We can use these inverse functions, along with right-triangle trigonometry (SOH CAH TOA) and the Pythagorean theorem, to solve for unknown angles and sides in both abstract problems and real-world scenarios.

KEY TERMS

  • Inverse Function: A function that "reverses" another function. For trig, if $\sin( heta) = x$, then $\sin^{-1}(x) = heta$.
  • Range: The set of all possible output values of a function. For inverse trig functions, this is a restricted set of angles.
  • Domain: The set of all possible input values. For inverse trig functions, this is a ratio, typically between -1 and 1.
  • Reference Angle: The acute angle formed by the terminal side of an angle and the x-axis.
  • Phase Shift: A horizontal shift of a periodic function.
  • Bearing: A direction measured from North or South. E.g., S 20° W means 20 degrees west of South.
A note on notation: $\sin^{-1}(x)$ is the inverse function. It is NOT a reciprocal: $\sin^{-1}(x) eq \frac{1}{\sin(x)}$. That's why your calculator has separate $x^{-1}$ and $\sin^{-1}$ buttons.

Core Formulas

  • Pythagorean Theorem: $a^2 + b^2 = c^2$ (or $x^2 + y^2 = r^2$)
  • Reciprocal Identity for Secant: $ \sec( heta) = \frac{1}{\cos( heta)} $

EXAMPLE 1: Matching a Tangent Graph

Problem: Match the graph of a tangent function that has been shifted. The period is $\pi$ (from 0 to $\pi$), and it passes through $(\pi/2, 0)$.

Analysis:

  1. Period: The period is $\pi$, so the $b$ value is 1. No period change.
  2. Vertical Shift: The graph's center point is on the x-axis, so there is no vertical shift ($d=0$).
  3. Phase Shift: The standard tangent graph passes through the origin $(0,0)$. This graph's center point has been shifted to $(\pi/2, 0)$. This indicates a phase shift of $\pi/2$ to the right. The formula for this shift is $(x - \pi/2)$.
  4. However, trig functions are periodic. A shift right by $\pi/2$ can sometimes be equivalent to a shift left by $\pi/2$. The equation could be $y = an(x - \pi/2)$ or $y = an(x + \pi/2)$. In this case, the provided option was $y = an(x + \pi/2)$, which represents a shift to the left by $\pi/2$. Both graph to the same picture.

Conclusion: There can be multiple correct equations for the same periodic graph. The given answer was $$y = an(x + \pi/2)$$

EXAMPLE 2: Composite Inverse Function (Unit Circle Value)

Problem: Find the exact value of $\sin^{-1}(\sin(5\pi/4))$.

Analysis:

  1. Start from the inside: Find $\sin(5\pi/4)$. The angle $5\pi/4$ is in Quadrant III, where sine is negative. The reference angle is $\pi/4$. So, $\sin(5\pi/4) = -\frac{\sqrt{2}}{2}$.
  2. Now the problem is $\sin^{-1}(-\frac{\sqrt{2}}{2})$.
  3. The range of $\sin^{-1}(x)$ is Quadrants I and IV. Since the input is negative, the answer must be in Quadrant IV.
  4. The angle in Quadrant IV that has a sine of $-\frac{\sqrt{2}}{2}$ is $-\frac{\pi}{4}$.

Answer: $$-\frac{\pi}{4}$$

The $\sin^{-1}$ and $\sin$ don't simply cancel out unless the original angle is within the restricted range (Quadrants I or IV). The calculator will always give you the correct answer in the proper range. If you typed this in, it would give you the decimal for $-\pi/4$.

EXAMPLE 3: Composite Inverse Function (Sketching a Triangle)

Problem: Find the exact value of $\cos(\sin^{-1}(6/10))$.

Analysis:

  1. Inner function: $ heta = \sin^{-1}(6/10)$. This means $\sin( heta) = 6/10$. Since the ratio is positive, $ heta$ is in Quadrant I.
  2. $\sin( heta) = \frac{ ext{opposite}}{ ext{hypotenuse}} = \frac{6}{10}$.
  3. Sketch a triangle in Q1 with opposite side = 6 and hypotenuse = 10.
  4. Find the adjacent side using Pythagoras: $a^2 + 6^2 = 10^2 \implies a^2 + 36 = 100 \implies a^2 = 64 \implies a = 8$.
  5. Outer function: The question is now "What is $\cos( heta)$?".
  6. Using the triangle: $\cos( heta) = \frac{ ext{adjacent}}{ ext{hypotenuse}} = \frac{8}{10}$.

Answer: $$\frac{8}{10} = \frac{4}{5}$$

EXAMPLE 4: Bearing Word Problem

Problem: A ranger at R1 sees a fire due south. A second ranger at R2, 6 miles east of R1, sights the same fire at a bearing of S 20° W. How far is the first ranger from the fire?

Analysis:

  1. Draw the setup. R1 is at some point. The Fire (F) is directly below it. R2 is 6 miles to the east, forming a right angle at R1.
  2. The bearing from R2 is S 20° W. This angle is between the south line from R2 and the line of sight to the fire.
  3. Inside the right triangle, the angle at R2 is the complement of the bearing angle: $90^\circ - 20^\circ = 70^\circ$.
  4. Let $d$ be the distance from R1 to F (the side opposite the $70^\circ$ angle). The adjacent side is the distance between rangers, 6 miles.
  5. Use tangent: $ an(70^\circ) = \frac{ ext{opposite}}{ ext{adjacent}} = \frac{d}{6}$.
  6. Solve for $d$: $d = 6 imes an(70^\circ)$.

Answer: $$d \approx 16.5 ext{ miles}$$
