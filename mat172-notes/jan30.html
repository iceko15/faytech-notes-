Advanced Trigonometric Transformations & Phase Shifts

BIG IDEA

While basic transformations involve amplitude and period changes, the introduction of the Phase Shift (horizontal displacement) requires a more sophisticated approach to grid scaling. By finding a common denominator (LCD) between the period increments and the phase shift, we can accurately plot complex trigonometric functions without "squishing" the graph or losing precision.

KEY TERMS

  • Amplitude ($A$): The vertical distance from the midline to the maximum or minimum. It is a vertical stretch/compression.
  • Period ($P$): The horizontal length of one full cycle. Calculated as $\text{Base Period} / B$.
  • Phase Shift ($C/B$): The horizontal shift of the graph. If positive, it moves right; if negative, it moves left.
  • Midline: The horizontal axis that the graph oscillates around.

STEPS/PROCESS: FINDING THE PHASE SHIFT

There are two primary methods to identify the phase shift. The professor recommends Method 1 for its intuitive handling of signs.

Method 1: The Zero-Equated Solve (Recommended)

  1. Take the entire argument (the "parentheses chunk") of the trig function: $Bx - C$.
  2. Set it equal to zero: $Bx - C = 0$.
  3. Solve for $x$. The result is your phase shift.
  4. Direction: A positive result means move Right. A negative result means move Left. (This follows standard number line logic).

Method 2: The Formulaic Approach

  • Identify $C$ and $B$.
  • Calculate $C/B$.
  • Warning: You must recognize that $C$ uses the opposite sign of what is seen in the parentheses. A minus sign in the chunk indicates a positive $C$ value.

RULES/IDENTITIES

Order of Transformations (MyMathLab & General Practice):

  1. Phase Shift (Horizontal Shift) - Always do this first if it exists.
  2. Stretching and Shrinking (Amplitude/Period).
  3. Reflections (Across X or Y axes).
  4. Vertical Shifts ($D$ value) - Always do this last.
"If you do these out of order in MyMathLab, it is highly likely it will count it wrong, even if your final picture looks exactly the same as theirs. Stick to the order of actions on page 24."

EXAMPLE 1: $y = 3\sin(2x - \pi/3)$

Step 1: Identify Constants

  • $A = 3$ (Amplitude: Height from midline is 3 units up/down).
  • $B = 2$ (Used for period).
  • Period $= 2\pi / 2 = \pi$.

Step 2: Calculate Phase Shift

Set $2x - \pi/3 = 0$

$2x = \pi/3$

$x = \pi/6$ (Shift Right because result is positive).

Step 3: Grid Planning (The LCD Issue)

We need a grid that accommodates both the quarters of the period ($\pi/4$) and the phase shift ($\pi/6$).

  • Period $= \pi$. Standard quarters would be $\pi/4$.
  • Phase Shift $= \pi/6$.
  • LCD of 4 and 6 is 12.
  • Recommendation: Let 1 box $= \pi/12$.

Step 4: Mapping the Graph (12 boxes = $\pi$)

  • Initial Period (no shift): 0 to $\pi$ (12 boxes).
  • Shifted Period: Start at $\pi/6$ (which is 2 boxes right). End at $7\pi/6$ (14 boxes from origin).
  • Critical Points (Quarters of 12 boxes $= 3$ boxes apart):
    • Start: $\pi/6$ (at midline)
    • 1st Quarter: $\pi/6 + 3$ boxes $= 5\pi/12$ (Max at 3)
    • Middle: $5\pi/12 + 3$ boxes $= 8\pi/12 = 2\pi/3$ (at midline)
    • 3rd Quarter: $8\pi/12 + 3$ boxes $= 11\pi/12$ (Min at -3)
    • End: $11\pi/12 + 3$ boxes $= 14\pi/12 = 7\pi/6$ (at midline)

EXAMPLE 2: MyMathLab Tool Usage

Function: $y = 4\cos(x - 2\pi)$

  • Amplitude: 4.
  • Period: $2\pi$ (since $B=1$).
  • Phase Shift: Solve $x - 2\pi = 0 \rightarrow x = 2\pi$.

MyMathLab UI Tips:

  • Select the specific tool (Sine vs. Cosine).
  • Click anywhere on the graph; it will "plop" a base model.
  • Use the pop-up menu to input values: Phase shift = 2pi, Amplitude = 4.
  • Pro-Tip: If you don't have a $\pi$ button, type "P-I" and the software should convert it to the symbol.

PRACTICE PROBLEMS / WARNINGS

"Some problems in the homework don't use $\pi$ in the grid. The grid might be 1, 2, 3, 4, 5 and the period might be 5. Don't treat these any differently! The logic remains the same regardless of the units."
  • Grammar Warning: In MyMathLab, if asked for the phase shift number, put "$\pi/6$". If it's a multiple choice asking about direction, select "Right $\pi/6$". Don't say "Left $-\pi/6$"; that's a double negative and bad grammar.
  • Tangent/Cotangent: The "squiggle" is oriented around the origin. If you can transform a Sine wave, you can transform these; the notion of shifting the "center" part is the same.

SUMMARY

Success in graphing complex trig functions relies on paper sketching before using digital tools. Identify $A, B, C$. Calculate the Phase Shift by solving the inner chunk for zero. Choose a grid scale based on the LCD of your quarter-period and your phase shift to ensure all critical points land on clean grid lines. Always perform the phase shift first and the vertical shift last.
