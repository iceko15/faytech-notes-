BIG IDEA

We often assume that inverse functions simply "cancel" each other out (e.g., $f^{-1}(f(x)) = x$). In Trigonometry, this is NOT always true due to domain restrictions. We must evaluate composite functions from the inside out, paying close attention to which quadrant the angle lands in versus which quadrant the inverse function is allowed to output.

KEY TERMS & RESTRICTIONS

  • Composite Function: A function inside another function, denoted as $f(g(x))$ or $(f \circ g)(x)$.
  • Inverse Sine ($\sin^{-1}$): Restricted to Quadrants I and IV $[-\frac{\pi}{2}, \frac{\pi}{2}]$.
  • Inverse Cosine ($\cos^{-1}$): Restricted to Quadrants I and II $[0, \pi]$.
  • Inverse Tangent ($\tan^{-1}$): Restricted to Quadrants I and IV $(-\frac{\pi}{2}, \frac{\pi}{2})$.

SCENARIO 1: SAME FUNCTION COMPOSITION

Does $\cos^{-1}(\cos(x)) = x$? Only if $x$ is in Quadrant I or II.

Example A: The "Easy" Case

Evaluate: $\cos^{-1}(0.7)$

Think of $0.7$ as $\frac{7}{10}$. This represents $\frac{\text{Adjacent}}{\text{Hypotenuse}}$. Since it is positive, we are in Quadrant I. The inverse exists and is valid.

Example B: The "Trap" Case

Evaluate: $\sin^{-1}(\sin(\frac{5\pi}{6}))$

1. Inside First: What is $\sin(\frac{5\pi}{6})$?

  • Reference angle is $\frac{\pi}{6}$.
  • Quadrant II (Sine is positive).
  • Value: $\frac{1}{2}$.

2. Outside Second: Evaluate $\sin^{-1}(\frac{1}{2})$.

  • We need an angle where sine is $\frac{1}{2}$, but it MUST be in the restricted range (Q1 or Q4).
  • Since $\frac{1}{2}$ is positive, we choose Q1.

Final Answer: $\frac{\pi}{6}$ (Note: It did not equal the original $\frac{5\pi}{6}$).

Example C: The Impossible Case

Evaluate: $\cos^{-1}(-1.2)$

Convert to a fraction: $-1.2 = -\frac{12}{10} = -\frac{6}{5}$.

This implies $\frac{\text{Adjacent}}{\text{Hypotenuse}} = \frac{6}{5}$.

Logic Check: The Hypotenuse (5) cannot be smaller than the Adjacent side (6). This triangle cannot exist.

Final Answer: Undefined / Not Possible.

SCENARIO 2: MIXED FUNCTION COMPOSITION

When the functions don't match (e.g., Sine of Inverse Tangent), we must draw a triangle.

Example D: Numerical Input

Evaluate: $\sin(\tan^{-1}(\frac{3}{4}))$

1. Inside (Setup Triangle): $\tan^{-1}(\frac{3}{4})$ implies a triangle in Q1 where:

  • Opposite = 3
  • Adjacent = 4

2. Find Missing Side: Pythagorean Theorem ($3^2 + 4^2 = c^2$).

  • $9 + 16 = 25 \rightarrow c = 5$ (Hypotenuse).

3. Outside (Evaluate): Find $\sin(\theta)$ for this triangle.

  • $\sin = \frac{\text{Opposite}}{\text{Hypotenuse}}$

Final Answer: $\frac{3}{5}$ (or 0.6).

Example E: Algebraic Input (Variable)

Evaluate: $\sec(\tan^{-1}(x))$ where $x > 0$.

1. Inside (Setup Triangle): $\tan^{-1}(x)$ can be written as $\tan^{-1}(\frac{x}{1})$.

  • Opposite = $x$
  • Adjacent = 1

2. Find Missing Side: Pythagorean Theorem.

  • $a^2 + b^2 = c^2 \rightarrow 1^2 + x^2 = c^2$
  • Hypotenuse = $\sqrt{x^2 + 1}$

3. Outside (Evaluate): Find $\sec(\theta)$.

  • $\sec = \frac{\text{Hypotenuse}}{\text{Adjacent}}$
  • $\sec = \frac{\sqrt{x^2 + 1}}{1}$

Final Answer: $\sqrt{x^2 + 1}$

SUMMARY

When dealing with inverse trigonometric functions, never assume they cancel. Always check the quadrant. If the input is a value (like 0.7 or 3/4), draw the triangle and solve for the missing side. If the input is a variable (like x), perform the same steps algebraically. For the upcoming test, utilize the Study Plan to practice specific question types you struggled with in the homework.
